# Happy Birthday Varsha

A birthday website for someone special.

**Live:** [arpithanms.github.io/Varsha_BDay](https://arpithanms.github.io/Varsha_BDay/)

## What Is This

Two experiences in one:

- **The Letter** — A scrollable, narrative-driven love letter with photos woven into the story
- **The Memories** — A full interactive site with photo galleries, video sections, a timeline, flip cards, sealed envelopes, a constellation map, and more

## Built With

- Single HTML files, no frameworks
- Vanilla CSS + JavaScript
- Google Fonts (Playfair Display, Lora, Dancing Script)
- Google Drive embeds for large videos
- GitHub Pages for hosting

## Made With Love

Built from scratch by [Arpithan](https://github.com/ArpithanMS) for Varsha's birthday.
